This project is a Vendor Management System developed during my internship at ECL.

It is a frontend web application built using HTML, CSS, and JavaScript. The system allows users to register, log in, and manage vendor details efficiently.

Key features include:
- User authentication (Login & Register)
- Add, view, and delete vendors
- Profile picture upload and management
- Data storage using browser LocalStorage

This project demonstrates my skills in frontend development, UI design, and JavaScript-based data handling.

Future improvements may include backend integration, database support, and advanced analytics dashboard.
